## 🌟 Project Overview

This is a modern, responsive website developed for **InAmigos Foundation** — a Central Government licensed Section 8 NGO established in 2020. The website serves as a comprehensive digital platform showcasing the organization's mission, ongoing projects, social impact statistics, success stories, and volunteer initiatives.

### Key Highlights
- **🎯 Mission**: Uniting minds for positive social change across India
- **📍 Coverage**: Operating across 28 states
- **👥 Impact**: 50,000+ lives impacted, 200+ volunteers, 20,000+ trees planted
- **🏆 Certifications**: 80G & 12A certified, CSR-1 registered, ISO 9001:2015 certified
- **🌱 Main Slogan**: "Every Pin Tells a Story of Impact"

## 🚀 Installation & Setup

### Prerequisites
- **Web Browser** (Chrome, Firefox, Safari, Edge)
- **Local Server** (Live Server extension, XAMPP, or similar)
- **Text Editor** (VS Code recommended)

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/inamigos-foundation.git
   cd inamigos-foundation
   ```

2. **Open the project**
   ```bash
   # Using VS Code
   code .
   
   # Or navigate to project directory
   cd inamigos-foundation
   ```

3. **Run with Live Server**
   - Install Live Server extension in VS Code
   - Right-click on `index.html`
   - Select "Open with Live Server"
   - Website will open at `http://localhost:5500`

4. **Alternative: Direct file access**
   ```bash
   # Open directly in browser
   open index.html  # macOS
   start index.html # Windows
   ```

### Development Setup

```bash
# For development with PHP features
# Using XAMPP/WAMP/MAMP
1. Copy project to htdocs/www folder
2. Start Apache server
3. Access via http://localhost/inamigos-foundation
```
